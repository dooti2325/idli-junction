# Idli Junction 🍛

Authentic South Indian cuisine crafted with tradition, served with love. Experience the finest idlis, dosas, and filter coffee.

## ✨ Features
- **Premium Design**: Modern, responsive UI with a rich South Indian aesthetic.
- **Dynamic Menu**: Real-time menu updates powered by Firebase.
- **Micro-animations**: Smooth scroll reveals, hover effects, and parallax backgrounds.
- **Authentic Content**: Handpicked images and descriptions reflecting true culinary heritage.

## 🛠️ Tech Stack
- **Frontend**: React.js, Vite, Tailwind CSS
- **Animations**: Framer Motion, Lucide React
- **Database**: Firebase Firestore
- **Deployment**: Vercel

## 🚀 Local Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file with your Firebase credentials (see `.env.example`).
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📸 Assets
The project uses high-quality professional photography for the food items and restaurant ambiance, located in the `public/images` directory.
